## Usage

Run the index.js file to run the program and to run the tests run poker.test.js with the following commands:

Note: [NodeJs](https://github.com/nodejs/node) was used to run the program and tests.

```bash
# Run program
node index.js

# Run tests
node poker.test.js
```
